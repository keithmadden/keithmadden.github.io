Typography Project

This project is it illustrate just how I have developed my design side. Using only text, black and white contrast and CSS skills, this two-page newspaper allowed me to show just how I have managed to come to terms with typography.


It was made with just HTML5 and CSS3, this doesn’t have any of your fancy elements like JavaScript that you might find in your average website.

The creation of each element, heading and paragraph was chosen for a reason. Showing the user just what I thought about and why I chose the font, style and weight.


Giving the user a feeling of a newspaper but the modern approach as a news website. The process of building this involved studying newspapers and their approach to design. It also holds some key elements such as line breaks that new modern websites are using
