Implementing a Reliable
Data Transfer Protocol

I built a fully functional simple reliable data transfer protocol. This protocol is able to send and receive data using the Layers that are in the current world wide web. Sending data between layer 5 (Session Layer) and passing the data down to Layer 3+4 (Protocol Layer).


The creation of each element, heading and paragraph was chosen for a reason. Built in the Java framework this application uses NetBeans.

Sending a message from one machine (port) and receiving the message on the other machine (port), in the background it has an acknowledgements function that lets me know that the data has been send, a check sum.


To let me know that the message hasn’t been lost or altered in any way and timer so if the package gets lost in transmission you will also be notified about it.
